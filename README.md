# 🌿 Hệ sinh thái tri thức | Môi trường - Xã hội - Nông nghiệp

Trang web tĩnh với giao diện tab động, cung cấp kiến thức chuyên sâu về:
- **Môi trường**: Biến đổi khí hậu, Khí nhà kính, Mô hình hóa, GIS & Cơ sở dữ liệu
- **Xã hội**: Cộng đồng bền vững, Công bằng môi trường, Giáo dục
- **Nông nghiệp**: Công nghệ cao, Sản xuất nông nghiệp bền vững

## 📋 Yêu cầu hệ thống

### Tùy chọn 1: Chạy với Node.js (Express)
- Node.js (phiên bản 14.x trở lên)
- npm (đi kèm với Node.js)

### Tùy chọn 2: Chạy với Python (Đơn giản hơn)
- Python 3.x

### Tùy chọn 3: Chạy trực tiếp trên trình duyệt
- Không cần cài đặt gì, chỉ cần trình duyệt web

## 🚀 Cách cài đặt và chạy

### Cách 1: Chạy với Node.js (Express) - Khuyến nghị

```bash
